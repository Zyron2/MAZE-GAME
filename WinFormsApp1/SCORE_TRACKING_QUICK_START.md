## ?? Controls

### During Game
```
Arrow Keys or WASD = Move player
```

### After Game Ends
```
R = Restart (New game)
H = High Scores (Top 10)
A = All Scores (Complete history)
S = Statistics (Game stats)
```

---

## ?? Two Files Are Maintained

### `highscores.txt` - Top 10 Only
- Auto-updated
- Keeps best 10 scores
- Traditional leaderboard

### `all_scores.txt` - Complete History
- ALL games recorded
- Never trimmed
- Complete tracking

   ?

   ?

   ?

   ?

## ?? What Gets Tracked

For each game:
```
? Player name
? Completion time (seconds)
? Date and time of play
```

**That's it!** Simple but complete!
