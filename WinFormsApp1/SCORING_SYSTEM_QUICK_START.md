# ?? SCORING SYSTEM - QUICK START

## ? NOW YOU CAN SAVE YOUR SCORES!

Your maze game now has a **working scoring system** with persistent high score tracking!

---

## ?? Quick Workflow

### 1. **Play Game**
```
Press F5 ? Navigate maze ? Reach exit
```

### 2. **Win!**
```
YOU WIN! screen appears
Shows your completion time
```

### 3. **Save Score**
```
Press N ? Enter your name ? Click OK
? Score saved automatically!
```

### 4. **View Scores Anytime**
```
Press H ? See Top 10 leaderboard
Press A ? See all games played
Press S ? See statistics
Press R ? Play another game
```

---

## ?? Controls After Game Ends

```
R = Restart (new game)
N = Save Score (if you won) ? NEW!
H = High Scores
A = All Scores
S = Statistics
```

---

## ?? What Gets Saved

For each win:
```
? Your name
? Completion time (seconds)
? Date and time
```
