## ?? Implementation Details

### What Happens When You Press N

1. **Check if you won**
   ```
   If Lost: Show warning "Can only save wins"
   If Won: Continue
   ```

2. **Prompt for player name**
   ```
   Dialog box appears:
   "Enter your name: [______]"
   OK / Cancel buttons
   ```

3. **Save the score**
   ```
   GameController.AddHighScore(playerName)
   ?
   HighScoreTable.AddScore(playerName, time)
   ?
   Saves to both files
   ?
   Confirmation message
   ```

4. **Show confirmation**
   ```
   "? Score saved!
   Player: {name}
   Time: {time}s"
   ```
